# .spectrum
dot spectrum is a pixel-art-like spectrum analyzer. w/ Web Audio API.


## 🛠️ Installation

1. yarn install

```
yarn
```

2. build localhost server

```
yarn static -p 8080
```

## 📚 Tech Stack
- Canvas API
- Web Audio API
- PWA
